# `culturabuilder pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
culturabuilder pairing list whatsapp
culturabuilder pairing approve whatsapp <code> --notify
```
